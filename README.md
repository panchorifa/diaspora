# Diaspora Stream

Python cli to retrieve diaspora stream for a given user

## CLI can be used to...

* Crawl stream posts
* Retrieve stats for the last 100 stream posts

### Examples

* python diaspora-cli.py stream
* python diaspora-cli.py stats
